1 总则
==============================================

.. raw:: html

  <h2 id="test111">1 总则</h2>

.. raw:: html

 <p style="text-align:justify"><a href="#id1.0.1"> 1.0.1</a> <span id="id1.0.1">加强城市桥梁的养护工作,提高城市桥梁的养护水平，维护城市桥梁设施，保障城市桥梁完好和安全运行，制定本标准。</span></p>
 <p style="text-align:justify"><a href="#id1.0.2"> 1.0.2</a> <span id="id1.0.2">标准适用于已竣工验收后交付使用的城市桥梁的养护。</span></p>
 <p style="text-align:justify"><a href="#id1.0.3"> 1.0.3</a> <span id="id1.0.3">市桥梁的养护除应符合本标准外，尚应符合国家现行有关标准的规定。</span></p>
