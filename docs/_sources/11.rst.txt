11 城市桥梁安全防护
==============================================

.. raw:: html

  <h2 id="test111">11 城市桥梁安全防护</h2>

11.1 城市桥梁安全保护区域
---------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id11.1.1"> 11.1.1</a> <span id="id11.1.1"> 城市桥梁养护应按结构形式、桥梁类型的不同划定城市桥梁安全保护区域范围，编制监督管理方案，发现桥梁安全隐患应及时进行处置。</span></p>
 <p style="text-align:justify"><a href="#id11.1.2"> 11.1.2</a> <span id="id11.1.2"> 在城市桥梁安全保护区域内可能影响城市桥梁安全的施工作业行为包括:</span></p>
 <ol>
 <li>河道疏浚、河道挖掘等施工作业。</li>
 <li>建筑打桩、修建地下结构物、盾构顶进、管线顶进、(架)埋设管线、爆破、基坑开挖、降水工程等作业。</li>
 <li>大面积堆物或减少载荷量超过20 kN/m²的作业。</li>
 <li>其他可能损害城市桥梁的作业。</li>   
 </ol>
 <p style="text-align:justify"><a href="#id11.1.3"> 11.1.3</a> <span id="id11.1.3"> 在城市桥梁安全保护区域内，从事可能影响城市桥梁安全的施工作业时，应制定城市桥梁安全保护设计方案和相应的施工方案，并签订城市桥梁安全保护协议。</span></p>
 <p style="text-align:justify"><a href="#id11.1.4"> 11.1.4</a> <span id="id11.1.4"> 在城市桥梁安全保护区域内，从事可能影响城市桥梁安全的施工作业时，对可能影响城市桥梁安全运行的，应由具有相应资质的专业检测单位进行桥梁结构检测，编制检测报告，并根据检测结果采取相应的加固措施。</span></p>
 <p style="text-align:justify"><a href="#id11.1.5"> 11.1.5</a> <span id="id11.1.5"> 应由具有相应资质的专业检测单位编制监测方案，施工作业期间，对相关城市桥梁进行动态监测，并定期报送城市桥梁动态监测记录。</span></p>
 <p style="text-align:justify"><a href="#id11.1.6"> 11.1.6</a> <span id="id11.1.6"> 对城市桥梁进行动态监测前，应根据使用情况、现有状态及设计要求制定其沉降、位移的监控值及报警值。</span></p>

11.2 超重车辆过桥
---------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id11.2.1"> 11.2.1</a> <span id="id11.2.1"> <b>超重车辆通过桥梁前，应经设计单位专项验算，必要时应进行加固设计和采取相应技术措施，并应详细记录存档。</b></span></p>
 <p style="text-align:justify"><a href="#id11.2.2"> 11.2.2</a> <span id="id11.2.2"> 超重车辆通过桥梁,应选用多轴多轮的运载车辆、选取桥梁技术状况较好、加固工程费用较低的路线通过。</span></p> 
 <p style="text-align:justify"><a href="#id11.2.3"> 11.2.3</a> <span id="id11.2.3"> 当超重车辆通过桥梁时,应符合下列规定:</span></p>
 <ol>
 <li>应临时禁止其他车辆过桥。</li>
 <li>沿桥梁的中心行驶,车速不得超过5 km/h。</li>
 <li>不得在桥上制动、变速、停留。</li>   
 </ol>
 <p style="text-align:justify"><a href="#id11.2.4"> 11.2.4</a> <span id="id11.2.4"> 当超重车辆通过桥梁时，应观测记录桥梁位移、变形、裂缝扩张，必要时,还应观测应力、应变等。</span></p>

11.3 桥 下 空 间
---------------------------
.. raw:: html

 <p style="text-align:justify"><a href="#id11.3.1"> 11.3.1</a> <span id="id11.3.1"> 桥下空间使用应满足城市桥梁安全需求，宜用于停放车辆、设置道路养护管理设施或进行绿化。</span></p>
 <p style="text-align:justify"><a href="#id11.3.2"> 11.3.2</a> <span id="id11.3.2"> 桥下空间使用单位应建立健全消防安全管理制度、环境卫生管理制度。</span></p> 
 <p style="text-align:justify"><a href="#id11.3.3"> 11.3.3</a> <span id="id11.3.3"> 当桥下搭建构筑物时,与桥梁底面、桥墩、桥台的距离不应少于1.5 m，且不得将桥墩、桥台封闭在内，同时应采取措施保护桥梁设施。</span></p>
 <p style="text-align:justify"><a href="#id11.3.4"> 11.3.4</a> <span id="id11.3.4"> <b>桥下空间使用不得影响城市桥梁日常养护、维修、检测作业。</b></span></p>
